Operators

- 할당연산자 = 값을 할당합니다 => let x = 10;
- 덧셈 연산자 +는 값을 더합니다 => let x=5; let y=2; let z=x+y 
- 곱셉 연산자 *는 값을 곱합니다 => let x=5; let y=2; let multi=x*y
- 비교 연산자 > 값을 비교합니다 => 변수를 선언후 값을 비교합니다
자바스크립트 연산자의 유형
산술연산자[+,-,*,**,/,%[나눈나머지값],++,--], 할당연산자, 비교연산자, 문자열연산자, 논리 연산자, 비트 연산자, 삼항 연산자

Logical Operators : 논리연산자


- && => 그리고 조건 두개가 일치 (and연산자)
- || => 둘중에 하나라도 맞으면 true (or연산자)
- ! => 같지 않음 부정이 진실이고 진실이 거짓이다. (난드연산자)
